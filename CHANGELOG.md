# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024

### Added
- Initial release of Car Photo Enhancer
- 5 professional presets for car photography (Gloss, Metallic, Deep Paint, Night Neon, Wheel Detail)
- Real-time photo editing with Instagram-style interface
- Automatic license plate blurring for privacy
- Reflection and glare removal tools
- Advanced color correction (brightness, contrast, saturation, temperature)
- Before/after comparison slider
- Edit history with undo functionality
- User authentication with Manus OAuth
- Photo storage and management
- Responsive design for mobile and desktop
- Download edited photos in high quality

### Features
- **Presets**: 5 car-specific filter presets
- **License Plate Blur**: Automatic detection and blurring
- **Reflection Removal**: Reduce glare and reflections
- **Color Correction**: Fine-tune colors and lighting
- **Before/After Slider**: Compare original and edited versions
- **Edit History**: Track and undo changes
- **Cloud Storage**: Save edited photos to cloud

### Technical
- React 19 + TypeScript
- Tailwind CSS 4 for styling
- tRPC for type-safe API calls
- MySQL database for storage
- Express.js backend
- Canvas API for image processing
