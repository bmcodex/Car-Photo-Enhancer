# Troubleshooting Guide

## Common Issues

### Database Connection Error
- Check DATABASE_URL
- Verify MySQL is running
- Check network connectivity

### Image Upload Fails
- Check file size
- Verify file format
- Check browser permissions

### Filters Not Applied
- Clear browser cache
- Try different browser
- Check JavaScript console

### Performance Issues
- Close other tabs
- Clear cache
- Use modern browser
- Check network speed
