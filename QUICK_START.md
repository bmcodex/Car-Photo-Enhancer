# Quick Start Guide

## 5-Minute Setup

1. Clone: `git clone https://github.com/bmcodex/Car-Photo-Enhancer.git`
2. Install: `pnpm install`
3. Configure: Copy `.env.example` to `.env.local`
4. Database: `pnpm db:push`
5. Run: `pnpm dev`

Visit http://localhost:3000

## First Edit

1. Sign in
2. Upload photo
3. Select preset
4. Download result
