# Frequently Asked Questions

## General

**Q: Is my data safe?**
A: Yes, we use HTTPS encryption and secure authentication.

**Q: Can I use this offline?**
A: No, you need internet connection for authentication and storage.

**Q: What image formats are supported?**
A: JPEG, PNG, and WebP.

## Features

**Q: Can I undo changes?**
A: Yes, use the Undo button in edit history.

**Q: How do I blur license plates?**
A: Use the License Plate tab in the editor.

**Q: Can I compare before and after?**
A: Yes, use the Compare tab with the slider.

## Technical

**Q: What browser should I use?**
A: Chrome, Firefox, Safari, or Edge (latest versions).

**Q: Is there a mobile app?**
A: Not yet, but the web app is mobile-friendly.

**Q: Can I download my photos?**
A: Yes, click the Download button.
